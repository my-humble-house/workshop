# 視覺設計規範：清水模毛玻璃（Concrete Glass）

日期：2026-06-10
適用範圍：本 repo 所有前台頁面（目前為 `index.html`）。**未來任何頁面調整都必須遵守本規範**；若需偏離，先更新本文件再改頁面。

## 設計概念

- 底圖改為冷灰石墨＋溫灰清水模的低彩度 CSS 單色漸層毛玻璃背景，不使用任何 SVG 繪圖或分割圖。
- 內容放在同樣取深冷灰的毛玻璃（frosted glass）卡片上，卡片用不對稱圓角打破方正感。
- 全站只使用下列冷灰石墨與溫灰清水模色票，不引入高彩度或品牌色。
- 內容文字、邊框、襯底、按鈕、統計長條都必須由下列色票衍生；可用同色 RGB 加透明度，但不可新增未列出的色相。

## 色彩 tokens

主色票：

冷灰石墨：`#090B0C` `#0E1214` `#151A1B` `#20292A` `#303634`

溫灰清水模：`#8B8D84` `#B7AA92` `#C8BDA6`

文字與亮色襯底：`#EDEAE2` `#15130F`

| Token | 值 | 用途 |
|---|---|---|
| `--ink` | `#EDEAE2` | **唯一內容文字色**（見文字顏色規則） |
| `--accent` | `#B7AA92` | 選中選項、主按鈕、進度條、標籤襯底 |
| `--accent-press` | `#C8BDA6` | 主按鈕 hover/按下 |
| `--on-accent` | `#15130F` | 亮色襯底上的文字 |
| `--bg` | `#0E1214` | 冷灰石墨頁面底色 |
| `--page-glass` | `linear-gradient(145deg, rgba(40,48,48,.75), rgba(21,25,26,.75), rgba(9,11,12,.75))` | 頁面大底圖，冷灰石墨單色漸層毛玻璃，75% 透明度 |
| `--glass` | `linear-gradient(145deg, rgba(43,44,39,.75), rgba(22,24,23,.75), rgba(9,11,12,.75))` | 內容卡片底圖，冷灰石墨混溫灰清水模，75% 透明度 |
| `--glass-solid` | `rgba(22,24,23,.75)` | 不支援 backdrop-filter 時的卡片退化底 |
| `--row` | `rgba(48,54,52,.78)` | 選項、輸入欄、統計軌道的襯底 |
| `--line` | `rgba(183,170,146,.40)` | 分隔線、卡片描邊、進度條軌道 |
| `--shadow` | `rgba(3,6,7,.68)` | 卡片投影 |
| 長條色階 | `#C8BDA6` `#B7AA92` `#8B8D84` `#687273` | 統計長條依名次取高對比低彩度色階 |

所有可見顏色必須來自上述色票或其 rgba 透明度版本。CSS mask 內不可見的黑白值不算視覺色彩，但不得用它們作為實際前景、背景或邊框色。

## 文字顏色規則（強制）

1. **所有內容文字一律使用同一個 `--ink`**——標題、內文、提示、進度文字、統計標籤、頁尾說明皆同色。不得使用淡灰、半透明或其他色相的文字做層級。
2. 唯二例外：
   - **有亮色襯底的文字**（選中選項、主按鈕、錯誤膠囊、部門標籤等填底元件）使用 `--on-accent`。
   - **placeholder** 使用 `--ink` 加 `opacity:.45`（同色降透明度，不是換色）。
3. 文字層級只能靠**字級、字重、字距（letter-spacing）**表達，不靠顏色。

## 字級層級

| 角色 | 字級 / 字重 / 其他 |
|---|---|
| 頁面主標 h1 | 24px / 700（行動 21px） |
| 題目標題 h2 | 20px / 700（行動 18px） |
| 區塊小標 h3（統計區） | 12px / 700 / letter-spacing .22em |
| 章節標籤（進度列左側） | 16px / 600 / letter-spacing .2em |
| 進度頁數（進度列右側） | 16px / 600 |
| 欄位標籤 | 16px / 600 |
| 內文、選項、輸入欄 | 16px |
| 提示 .hint | 14px（統計區內 .hint 例外為 16px） |
| 統計區文字、統計列、任務列表、部門標籤 | 16px |
| 頁尾說明、錯誤膠囊 | 13px |
| 標籤膠囊（部門） | 11px / 600 |

最小字級 11px，不得更小。

## 版面與背景底圖

- 頁面底色 `--bg`，body 置中單欄，內容最大寬 620px。
- 背景底圖只能用 CSS `body::before` 產生：使用冷灰石墨＋溫灰清水模的單一 `linear-gradient` 做 75% 透明度毛玻璃背景，不使用分割斜面、色塊切線或材質網格。
- 禁止 inline SVG、外部 SVG、data URI SVG、canvas 幾何繪圖作為背景或圖示。需要方向符號時用 CSS pseudo-element。
- 背景層 `position:fixed`、`pointer-events:none`、z-index 0；內容卡片 z-index 1。
- 背景可見色一律從上述色票取用，不得引入米白、紅色、藍色或高彩度色相。

## 毛玻璃卡片

- 底 `--glass` ＋ `backdrop-filter:blur(18px)`（含 `-webkit-` 前綴），透明度維持 75%。
- 描邊 `1px solid --line`，陰影 `0 22px 46px rgba(42,46,44,.22)`。
- **不對稱圓角**：桌面 `30px 30px 76px 30px`（右下角放大）；行動 `24px 24px 52px 24px`。
- 以 `@supports not (backdrop-filter…)` 退化為 `--glass-solid` 半透明底。

## 元件規範

- **選項（選單類，可保持方正）**：襯底 `--row`、圓角 11px、描邊 `--line`；hover 描邊 `--accent`；**選中＝整塊反白成 `--accent` 填底＋ `--on-accent` 文字**，不用打勾圖示。
- **輸入欄／textarea**：同選項襯底與圓角，focus 描邊 `--accent`。
- **按鈕**：膠囊形（`border-radius:99px`）。主按鈕 `--accent` 填底；次要按鈕無底無框、`--ink` 文字、hover 底線。
- **進度條**：4px 高、軌道 `--line`、填色 `--accent`、兩端圓頭；上方左側章節標籤、右側頁數文字（`07／15`）皆為 16px，包含通行碼鎖定頁。
- **錯誤提示**：同色票填底膠囊（`--accent` 底＋`--on-accent` 字），不用紅色。
- **統計長條**：18px 高圓頭膠囊，軌道 `--row`，填色依名次取長條色階（超過第 4 名沿用最淺）。
- **部門標籤**：同色票膠囊（`--accent` 底、`--on-accent` 字、11px）。
- **焦點樣式**：`outline:2px solid --accent; outline-offset:2px`，不得移除。
- 動畫尊重 `prefers-reduced-motion`。

## 禁止事項

- 禁止回到「白卡片＋彩色主題色」的舊樣式；禁止引入冷灰石墨＋溫灰清水模色票以外的色相（含紅色錯誤字）。
- 禁止使用任何 SVG 繪圖作為背景、圖示或裝飾。
- 禁止用顏色深淺區分文字層級（見文字顏色規則）。
- 其餘技術邊界（原生 HTML/CSS/JS、禁框架、禁 icon font、禁外部字型）見 `CLAUDE.md`。
