# 視覺設計規範：寒舍集團內部表單（MHH Internal Form）

日期：2026-06-10（2026-06-11 雙欄版面修訂）
適用範圍：本 repo 所有前台頁面（目前為 `index.html`）。**未來任何頁面調整都必須遵守本規範**；若需偏離，先更新本文件再改頁面。

## 設計概念

- 視覺參考寒舍集團官網的黑白 logo、深藍灰導覽色、沉穩綠按鈕色與大面積留白，轉譯為內部表單風格。
- 問卷不得呈現常見 AI 產物式深色玻璃卡、漸層霓虹、圓角膠囊堆疊或過度科技感。
- 全站只使用下列 MHH 內部色票，不引入高彩度或未列出的品牌色。
- 內容文字、邊框、襯底、按鈕、統計長條都必須由下列色票衍生；可用同色 RGB 加透明度，但不可新增未列出的色相。

## 色彩 tokens

主色票：

Logo / 文字黑白：`#050505` `#FFFFFF`

官網式中性色：`#2C2A24` `#666666` `#F6F6F6` `#EFEAE1`

集團輔助色：`#3B5D6B` `#8FAF9B` `#81603F`

| Token | 值 | 用途 |
|---|---|---|
| `--ink` | `#2C2A24` | **唯一內容文字色**（見文字顏色規則） |
| `--accent` | `#3B5D6B` | 選中選項、進度條、標籤襯底 |
| `--accent-press` | `#284450` | 主按鈕 hover/按下 |
| `--on-accent` | `#FFFFFF` | 深色襯底上的文字 |
| `--bg` | `#F6F6F6` | 官網式淺灰頁面底色 |
| `--paper` | `rgba(255,255,255,.94)` | 主要表單紙面 |
| `--row` | `rgba(255,255,255,.86)` | 選項、輸入欄、統計軌道的襯底 |
| `--line` | `rgba(59,93,107,.24)` | 分隔線、卡片描邊、進度條軌道 |
| `--line-strong` | `rgba(59,93,107,.52)` | hover 或重點邊線 |
| `--shadow` | `rgba(44,42,36,.12)` | 紙面投影 |
| 長條色階 | `#3B5D6B` `#8FAF9B` `#81603F` `#666666` | 統計長條依名次取集團低彩度色階 |

所有可見顏色必須來自上述色票或其 rgba 透明度版本。CSS mask 內不可見的黑白值不算視覺色彩，但不得用它們作為實際前景、背景或邊框色。

## 文字顏色規則（強制）

1. **所有內容文字一律使用同一個 `--ink`**——標題、內文、提示、進度文字、統計標籤、頁尾說明皆同色。不得使用淡灰、半透明或其他色相的文字做層級。
2. 唯二例外：
   - **有亮色襯底的文字**（選中選項、主按鈕、錯誤膠囊、部門標籤等填底元件）使用 `--on-accent`。
   - **placeholder** 使用 `--ink` 加 `opacity:.45`（同色降透明度，不是換色）。
3. 文字層級只能靠**字級、字重、字距（letter-spacing）**表達，不靠顏色。

## 字級層級

| 角色 | 字級 / 字重 / 其他 |
|---|---|
| 頁面主標 h1 | 24px / 700（行動 21px） |
| 題目標題 h2 | 20px / 700（行動 18px） |
| 區塊小標 h3（統計區） | 12px / 700 / letter-spacing .22em |
| 章節標籤（進度列左側） | 16px / 600 / letter-spacing .2em |
| 進度頁數（進度列右側） | 16px / 600 |
| 欄位標籤 | 16px / 600 |
| 內文、選項、輸入欄 | 16px |
| 提示 .hint | 14px（統計區內 .hint 例外為 16px） |
| 統計區文字、統計列、任務列表、部門標籤 | 16px |
| 頁尾說明、錯誤膠囊 | 13px |
| 標籤膠囊（部門） | 11px / 600 |

最小字級 11px，不得更小。

## 版面與背景

- **電腦版面（雙欄）**：採用 grid 佈局（`.app`），左側 `300px` 固定側欄（`.side`，深藍底紙白字），右側內容區（`.main`，寬度自適應），內容卡片限制最大寬 `680px`。
- **行動版面（單欄，斷點 900px）**：當螢幕寬度小於 `900px` 時自動折疊，左側側欄轉變為頂部水平導覽列，大綱進度清單自動隱藏，卡片寬度自適應以符合窄螢幕閱讀。
- **背景與漸層**：內容區採用 CSS 漸層毛玻璃底色，不使用分割斜面、色塊切線、材質網格或科技感背景。
- **禁止裝飾**：禁止使用 inline SVG、外部 SVG、data URI SVG、canvas 幾何繪圖作為背景或圖示。需要方向符號時用 CSS pseudo-element。
- 背景層 `position:fixed` 或配合 grid 自適應，不使用高彩度或未列出之米白、紅色、亮藍、紫色。

## 內部表單紙面

- **深色側欄**：底色 `#142028`，邊框線與打勾圈採用清水模綠 `#9FB8AB` 強調。
- **表單內容卡片**：底 `#FFFFFF`（紙白），描邊 `1px solid --c-line`，陰影採用極輕量級偏深投影（`--shadow-card`）。
- **圓角克制**：桌面與行動皆 `2px - 5px`；不得使用大型膠囊、超大圓角或不對稱圓角。
- **品牌 Logo**：表單左側與側欄頂端必須使用 repo-local 官方 logo PNG：`assets/mhh-logo.png`，搭配 `CLAUDE 工作坊`、`課前調查` 品牌識別。不得外部熱連，不使用 SVG logo。

## 元件規範

- **選項（選單類／主題卡）**：底色為 `--c-paper`、描邊 `--c-line`，圓角 2px，具有微投影。選中後邊框變 `--c-accent`（官網藍）、底色變為 `--c-accent-soft`（淺藍灰），外加選中光暈 `--glow-accent`。滑鼠 hover 時卡片微浮，投影加深。
- **輸入欄／textarea**：同選項紙白底與 2px 圓角，focus 描邊 `--c-accent`，陰影 `--glow-focus`。
- **按鈕**：2px 圓角。主按鈕採用官網藍填底（`--c-accent`），hover 時變深按鈕藍（`--c-accent-deep`）；次要按鈕無底無框、墨色文字、hover 底線。
- **進度條**：電腦版位於側欄底部，高度 3px，填色為清水模綠 `--c-sage`，帶有綠光效果 `--glow-sage`；行動版位於頂列底部。
- **錯誤提示**：同色票填底方角標籤（`--c-accent-soft` 底＋`--c-accent-deep` 字），不用紅色。
- **統計長條**：10px - 18px 高，自選長條填色為官網藍 `--c-accent`，診斷長條填色為清水模綠 `--c-sage`。
- **講師啞鈴圖**：在統計進度軌上，學員自選以實心圓圈呈現，系統診斷以空心圓圈呈現，兩者相連線條為 `--c-line-strong`，重合時顯示雙色重疊圓圈。
- **資安暴露警示卡**：採用穩重棕金底色 `--c-bronze` 字白，不使用紅色。
- **焦點樣式**：`outline:2px solid --c-accent; outline-offset:2px`，不得移除。
- 動畫尊重 `prefers-reduced-motion`。

## 禁止事項

- 禁止回到「深色 AI 玻璃卡＋膠囊按鈕＋科技漸層」的舊樣式；禁止引入 MHH 內部色票以外的色相（含紅色錯誤字）。
- 禁止使用任何 SVG 繪圖作為背景、圖示或裝飾。
- 禁止用顏色深淺區分文字層級（見文字顏色規則）。
- 其餘技術邊界（原生 HTML/CSS/JS、禁框架、禁 icon font、禁外部字型）見 `CLAUDE.md`。
