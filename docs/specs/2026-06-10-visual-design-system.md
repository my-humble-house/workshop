# 視覺設計規範：清水模毛玻璃（Concrete Glass）

日期：2026-06-10
適用範圍：本 repo 所有前台頁面（目前為 `index.html`）。**未來任何頁面調整都必須遵守本規範**；若需偏離，先更新本文件再改頁面。

## 設計概念

- 底色為加深的清水模灰，上面鋪一層「密集但融入」的幾何線條元素。
- 內容放在毛玻璃（frosted glass）卡片上，卡片用不對稱圓角打破方正感。
- 全站單色階（炭墨灰）跳色，不引入其他彩度。
- 幾何元素、卡片、選項襯底全部使用同一個米白色系的不同透明度，達成「視覺上有線條、但融為一體」。

## 色彩 tokens

| Token | 值 | 用途 |
|---|---|---|
| `--ink` | `#242826` | **唯一的文字深色**（見文字顏色規則） |
| `--accent` | `#2B2F2D` | 炭墨跳色：選中選項、主按鈕、進度條、標籤襯底、統計第一名長條 |
| `--accent-press` | `#1A1D1C` | 主按鈕 hover/按下 |
| `--on-dark` | `#EFF1EE` | 深色襯底上的文字 |
| `--bg` | `#A8ABA6` | 頁面底色（深清水模灰） |
| `--glass` | `rgba(244,245,243,.55)` | 毛玻璃卡片底 |
| `--glass-solid` | `rgba(244,245,243,.93)` | 不支援 backdrop-filter 時的卡片退化底 |
| `--row` | `rgba(255,255,255,.6)` | 選項、輸入欄、統計軌道的襯底 |
| `--line` | `rgba(255,255,255,.7)` | 分隔線、卡片描邊、進度條軌道 |
| 長條色階 | `#2B2F2D` `#4A4F4C` `#6B6F6B` `#8A8D88` | 統計長條依名次由深到淺 |

幾何背景元素一律使用 `#F4F5F3`（即卡片同色），只靠 opacity（約 .14–.55）做層次。**禁止**在幾何元素上使用其他色相。

## 文字顏色規則（強制）

1. **所有文字一律使用同一個深色 `--ink`**——標題、內文、提示、進度文字、統計標籤、頁尾說明皆同色。不得使用淡灰、半透明或其他色相的文字做層級。
2. 唯二例外：
   - **有襯底的文字**（選中選項、主按鈕、錯誤膠囊、部門標籤等深色填底元件）使用 `--on-dark`。
   - **placeholder** 使用 `--ink` 加 `opacity:.45`（同色降透明度，不是換色）。
3. 文字層級只能靠**字級、字重、字距（letter-spacing）**表達，不靠顏色。

## 字級層級

| 角色 | 字級 / 字重 / 其他 |
|---|---|
| 頁面主標 h1 | 24px / 700（行動 21px） |
| 題目標題 h2 | 20px / 700（行動 18px） |
| 區塊小標 h3（統計區） | 12px / 700 / letter-spacing .22em |
| 章節標籤（進度列左側） | 11px / 600 / letter-spacing .2em |
| 進度大數字 `.p-num` | 26px / 600（總題數 `.p-total` 13px） |
| 內文、選項、輸入欄 | 16px |
| 提示 .hint、統計列 | 14px |
| 頁尾說明、錯誤膠囊 | 13px |
| 標籤膠囊（部門） | 11px / 600 |

最小字級 11px，不得更小。

## 版面與背景幾何

- 頁面底色 `--bg`，body 置中單欄，內容最大寬 620px。
- 幾何背景為一個 `position:fixed` 滿版 inline SVG（`.geo`，`aria-hidden="true"`，`pointer-events:none`，z-index 0），內容卡片 z-index 1。
- 幾何語彙限定：正圓環（同心）、正三角形（實心淡面＋描邊錯位重影）、45° 旋轉方形、平行斜線（兩條一組、透明度遞減）、散點小圓；另可加 1–2 個大圓做 `feGaussianBlur` 柔化光斑。
- 所有幾何元素描邊 1.5px、色 `#F4F5F3`，透明度區間 .14–.55。

## 毛玻璃卡片

- 底 `--glass` ＋ `backdrop-filter:blur(18px)`（含 `-webkit-` 前綴）。
- 描邊 `1px solid --line`，陰影 `0 22px 46px rgba(42,46,44,.22)`。
- **不對稱圓角**：桌面 `30px 30px 76px 30px`（右下角放大）；行動 `24px 24px 52px 24px`。
- 以 `@supports not (backdrop-filter…)` 退化為 `--glass-solid` 實色底。

## 元件規範

- **選項（選單類，可保持方正）**：襯底 `--row`、圓角 11px、無邊框；hover 描邊 `--accent`；**選中＝整塊反白成 `--accent` 填底＋ `--on-dark` 文字**，不用打勾圖示。
- **輸入欄／textarea**：同選項襯底與圓角，focus 描邊 `--accent`。
- **按鈕**：膠囊形（`border-radius:99px`）。主按鈕 `--accent` 填底；次要按鈕無底無框、`--ink` 文字、hover 底線。
- **進度條**：4px 高、軌道 `--line`、填色 `--accent`、兩端圓頭；上方左側章節標籤、右側兩位數大數字（`07／15`）。
- **錯誤提示**：炭墨填底膠囊（`--accent` 底＋`--on-dark` 字），不用紅色。
- **統計長條**：18px 高圓頭膠囊，軌道 `--row`，填色依名次取長條色階（超過第 4 名沿用最淺）。
- **部門標籤**：炭墨膠囊（`--accent` 底、`--on-dark` 字、11px）。
- **焦點樣式**：`outline:2px solid --accent; outline-offset:2px`，不得移除。
- 動畫尊重 `prefers-reduced-motion`。

## 禁止事項

- 禁止回到「白卡片＋彩色主題色」的舊樣式；禁止引入炭墨灰階以外的色相（含紅色錯誤字）。
- 禁止用顏色深淺區分文字層級（見文字顏色規則）。
- 其餘技術邊界（原生 HTML/CSS/JS、禁框架、禁 icon font、禁外部字型）見 `CLAUDE.md`。
