課前問卷改版：三個方向

針對你提的四個問題——版型呆板、文字重複、下一步引導弱、統計不直覺——以下是共同的修正，再加上三個視覺與版面方向供比較。配色取自 mhh-group.com 的深藍灰、沉穩綠與大量留白。

問題 → 對策

  • Logo 顯示不出來：原檔是白色透明底（官網深色區用），放在白卡上等於隱形。方向 A、C 改用深色底直接放白 logo；方向 B 將 logo 重新上色為墨色版。
  • 文字重複：通行碼頁併入歡迎頁，同一句話只出現一次；題目頁移除與進度列重複的部分標籤。
  • 引導回饋弱：選中有勾選動效、單選自動前進（可關）、換部分有過場頁、分段進度條、送出有確認動畫。
  • 統計不直覺：分兩層——學員看大數字＋主題排行＋任務牆；講師切換「講師檢視」看自選 vs 診斷啞鈴圖與備課分布。

流程也重排了

  • 歡迎頁列出五個部分與題數，先給全貌再開始。
  • 選擇題維持一頁一題；課堂實作四題合併成一頁（同一個案例的四個面向），總頁數變少。
  • 「去識別」的長說明改為點擊展開，不再佔半頁。
  • 統計為原型假資料；正式版接回原本的 Worker API 即可，匿名彙整邏輯未變。

三個方向

點「開啟完整原型」可從頭到尾互動填寫；右上工具列開啟 Tweaks 可換強調色、開關自動前進。

方向 A

官網沉穩版

最貼近 mhh-group.com 的氣質：深藍黑表頭帶＋白色官方 logo、襯線標題、紙面卡片。

  • 換部分時整卡反轉為深色過場，節奏感最強
  • 適合想維持集團正式調性的場合

方向 B

輕快內部版

內部工具的友善感：墨色 logo＋淺色紙面、沉穩綠強調、圓角與膠囊按鈕，整體更輕。

  • 視覺最不嚴肅，填起來像聊天不像考試
  • 適合想降低「被調查感」的場合
開啟完整原型
方向 C

桌面雙欄版

學員多用電腦，所以把進度變成常駐的深色側欄：五個部分打勾推進，永遠知道自己在哪。

  • 白 logo 放側欄，品牌與導覽合一
  • 定位感最好，統計頁空間也最寬裕

重點畫面並排
